Data info:

6 non-DWIs (b = 0 s/mm^2)
60 DWIs (b = 1200 s/mm^2)
voxel size = 1 x 1 x 1 mm^3
SNR = inf (no noise added)


This phantom data set is created as described in:
Leemans A, Sijbers J, Verhoye M, Van der Linden A, Van Dyck D
"Mathematical framework for simulating diffusion tensor MR neural fiber bundles"
Magnetic Resonance in Medicine 2005, Vol. 53, No. 4, p. 944-53


As an example, this phantom data set has been used to investigate tractography properties in:
Jones DK
"Challenges and limitations of quantifying brain connectivity in vivo with diffusion MRI"
Imaging in Medicine 2010, Vol. 2, No. 3, p. 341-355
